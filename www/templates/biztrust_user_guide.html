USER GUIDE
1. Target Users

Anybody individual or company who wishes to verify the validity of the copy BizTrust info from Suruhanjaya Syarikat Malaysia (SSM)

2. How the apps work

i. The SSM BizTrust QR Code screen will be displayed. Touch the "Tap screen to scan" to start scanning the QR Code
ii. After the QR Code scanner start, user is able to point the scanner window towards the QR code. Tap button "x" located at a convenient position to redirect to the front page.

3. Output

i. Corporate informatin will be displayed if the QR code scanning is successful. Tap QR icon located at a convenient position to redirect to SSM BizTrust QR code page.
ii. Apps will display "Invalid SSM BizTrust QR Code" if the QR Code is not valid. Tap QR Code icon to redirect to SSM BizTrust QR code page.
iii. Apps will display "Information Not Found" if the entity has not registered under SSM. Tap QR code icon to redirect to SSM BizTrust QR code page.

4. Verify

Users have to user MySSM apps to verify the BizTrust QR code.
